About me

I am a student in my third year of Applied Computer Science at a university in Krakow. Prior to that, I graduated from a technical high school with the title of technician.

In my free time, I enjoy computer games, exploring new IT technologies and hardware. As a computer science student, I have developed skills in analyzing and interpreting complex data sets. I also enjoy developing software projects and experimenting with different coding techniques and frameworks to improve my skills. Furthermore, I have an interest in cybersecurity and enjoy learning about different security practices and methods to protect data and systems from cyber threats. I believe that understanding cybersecurity is crucial in today's digital age, and I am constantly seeking opportunities to expand my knowledge in this area. I believe that it is important to maintain a balance between work and play, as it helps me to stay creative, focused, and energized.

If you're interested in connecting with me or learning more about what I do, please feel free to reach out. I am always open to new ideas and collaborations!

Recent Projects

Weather Prophet

C# | .NET Core | Newtonsoft.Json

GitHub repository

The goal of the project is to provide users from around the world with a multilingual desktop application that allows them to check the current weather and forecast for the next few days for a specific city.

Pomodoro Timer

C# | .NET Core | Newtonsoft.Json

GitHub repository

Usefull tool that helps users apply the Pomodoro technique by providing a timer for the work and break sessions, as well as a list of completed sessions.

Hotel Las Palmas

C# | SQL | .NET Core | Entity Framework

GitHub repository

Hotel room reservation system that includes features like managing client databases, adding, editing and deleting existing client information, data validation and cost calculation for renting specific rooms for a certain number of days.

Intel8086

C# | .NET Core

GitHub repository

A simple INTEL 8086 processor command simulator created for executing basic commands like MOV or XCHG. It contains functions to reset and properly fill registers with random values.

Rube Goldberg Machine

C# | UnityEngine | .NET Framework | Unity3D

GitHub repository

An engineering project involving a machine that performs simple tasks in an execution way. The machine, which is only in the force of gravity, kinetics, magnetism and electricity, will guide the balls through a series of pipes, tunnels, a ramp and other activities to reach the goal.
